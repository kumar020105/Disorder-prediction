# Disorder Prediction Using Deep Learning Models

## Overview
This project predicts disorders based on medical input parameters using three deep learning models: Multi-Layer Perceptron (MLP), Artificial Neural Network (ANN), and Convolutional Neural Network (CNN).

## Objective
To compare the performance of MLP, ANN, and CNN models for disorder prediction and analyze their prediction accuracy.

## Technologies Used
- Python
- Google Colab
- TensorFlow
- Keras
- NumPy
- Pandas
- Scikit-Learn
- Matplotlib

## Models Implemented
1. Multi-Layer Perceptron (MLP)
2. Artificial Neural Network (ANN)
3. Convolutional Neural Network (CNN)

## Workflow
- Data Collection
- Data Preprocessing
- Feature Selection
- Model Training
- Model Evaluation
- Prediction Generation

## Results
The trained models predict the disorder category based on the input medical parameters. Performance metrics such as accuracy, precision, recall, and F1-score were used for evaluation.
